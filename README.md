1. 计划分为3步  `关于软考`  `前端学习`  `面试准备`
2. 每周末都需要写 当期日期的`目前状况`和`未来计划`
3. 上班有时间就写每日一题
4. 每天晚上都要学习和记录
5. 28号开始 记录每天时间，需要总结和规划时间
